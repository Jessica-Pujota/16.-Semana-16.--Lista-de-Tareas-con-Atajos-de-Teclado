# 16.-Semana-16.--Lista-de-Tareas-con-Atajos-de-Teclado

## Atajos de Teclado (Power User)
Para mejorar la eficiencia, se han implementado los siguientes bindings:
* **[Enter]**: Registra la tarea escrita en el campo de texto.
* **[C]**: Marca la tarea seleccionada como completada (se tornará color verde).
* **[Delete] / [D]**: Elimina la tarea seleccionada de la lista.
* **[Escape]**: Cierra la aplicación de forma segura.
